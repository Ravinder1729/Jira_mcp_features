# Jira & GitHub AI Assistant (CognifyCode)

## Overview
This application integrates Jira and GitHub to automate agile workflows.  
It generates Jira user stories from PRDs and validates implementation progress by correlating Jira issues with GitHub commits.

---

## Environment Variable Configuration

The application requires valid API tokens to connect securely to Jira, GitHub, and the AI provider.

### Required Environment Variables
Set the following environment variables before running the application:

```env
JIRA_API_TOKEN=your_jira_api_token
GITHUB_API_TOKEN=your_github_api_token
GEMINI_API_KEY=your_gemini_api_key
